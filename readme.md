## :toolbox: Getting Started

### :gear: Installation


install dart for windows
```bash
choco install dart-sdk
```

install dart for macos
```bash
brew tap dart-lang/dart brew install dart
```

get dependencies
```bash
dart pub get
```



### :running: Run Locally

Clone the project

```bash
https://github.com/maartenwilloque/APFront
```

Go to project directory
```bash
cd APfront
```

Run the app
```bash
flutter run
```
